题目编号竞赛题目任务说明数据集说明
#1AI技术趋势分析与展示1. 针对提供的ArXiv网站论文集数据，编写程序自动遴选出AI/ML主题的论文集合
2. 针对遴选出的AI/ML主题的论文集合进行分析，编写程序自动提取AI/ML相关的技术关键词，如“deep learning”或“random forest”等
3. 编写程序分析并采用数据可视化方法展示上述AI/ML技术关键词，随时间变化的趋势		 数据集由包含AI关键字的数千篇arXiv.org网站论文(pdf 格式）组成，论文含有题目、作者、学校机构、摘要和发表日期等
#2开源软件项目主题自动提取1. 在提供的开源软件数据集中挑选某个特定开源软件项目，通过程序计算出其主题
2. 编写程序，在提供的开源软件数据集中搜寻与该开源软件项目类似主题的项目
3. 编写程序分析并以数据可视化方式展示上述项目的主题相似度		 数据集包含多个常用Java开源项目源码
#3代码审查候选人推荐1. 作为软件质量保证的关键环节，代码评审是软件研发过程中的一个极为重要的环节
2. 在现代软件开发过程中，代码评审组织一般需要人为地指定评审者（评审者应具备足够的技术能力来完成该工作）
3. 本项目希望能从提供的代码评审历史数据集中，编写程序自动分析出某特定评审任务所需技能，并推荐候选评审者		 数据集由代码审核工具Gerrit 产生，包含3个大型基础软件开源项目(Android, OpenStack和Qt)的代码评审历史数据
#4软件缺陷自动预测1. 编写程序分析所提供的软件缺陷与其它软件工程度量数据集，建立软件缺陷与其它度量参数之间的量化模型
2. 采用上述模型，编写程序对某个软件项目进行自动缺陷预测		 每组数据集存放在独立的arrf文件里，有详细属性 (attribute) 说明, 第1个（total_loc）到倒数第2个（formal_parameters），都是特征（feature）
最后一个属性是目标 (target), 是要预测的属性，在本任务中是defects, 可选值为 true 和 false
#5AI领域专利结构分析1. 编写程序分析提供的AI领域专利数据集，找出专利间的引用与依赖关系
2. 以数据可视化方法展示上述关系		 该数据集包含数百个人工智能领域的专利信息。每个专利信息存于一个json文件。文件名由两部分组成，并通过“_”分割。
前半部分是专利的名称，后半部分是专利的公开时间，格式为yyyy年mm月dd日。如果这一数值为-1，表示该专利目前尚未显示开放日期。
专利信息是1维数组，长度为8，包含名称，摘要，公开时间（-1表示数据为空），专利网站链接，专利发明人（复数），专利所有人，依赖专利（均为空），专利号"
#6问答网站语义相似度分析1. 使用给定的Quora问题数据集，通过机器学习方法挖掘数据特征并搭建出语义相似度模型
2. 使用测试数据集评测模型效果
3. 鼓励尝试使用多种机器学习算法分别搭建模型并进行参数调优以获得最优模型		 请仔细阅读本任务的数据集根目录下的README文件，它包含详细说明
